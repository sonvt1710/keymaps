# Zenless Zone Zero Keymap [Aspect Ratio: 16:10] [Application bundle ID: com.HoYoverse.Nap]

**<ins>NOTE: This is a multi-level keymap, hence you will download and import all three post 2.0 keymaps.</ins>**. You can switch keymaps by pressing `Command (⌘)` + `[` or `Command (⌘)` + `]` in game.

## City (com.HoYoverse.Nap-Post2.0-City[16-10].playmap)
This keymap is supposed to be used when exploring the city, like when in Sixth Street or Failume Heights

W/A/S/D → Movement<br />
Esc → Pause/Home<br />
C → Agents<br />
F1 → Events<br />
F2 → Compendium<br />
F3 → New Eridu City Fund<br />
F4 → Signal Search<br />
Tab → Shortcut Wheel<br />
F → Interact<br />
T → Switch Characters/Time Periods<br />
L → Photos<br />
M → Map<br />
N → Quick Navigation<br />
V → Comission Tracking<br />
X → Fairy Auto Explore<br />
J → Inter-Knot<br />
RMB/Lshift → Sprint<br />

## Combat (com.HoYoverse.Nap-Post2.0-Combat[16-10].playmap)
This keymap is supposed to be used when fighting in Shiyu Defense, farming for character mats, and doing Combat commisions via the HDD

W/A/S/D → Movement<br />
Esc → Pause/Home<br />
LMB → Basic Attack<br />
Q → Ultimate<br />
E → Special Attack<br />
1 → Chain Attack - Left<br />
2 → Chain Attack - Right<br />
RMB/Lshift → Dodge<br />
Space → Switch to next character<br />
